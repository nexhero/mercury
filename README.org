* Description
Mercury is an peer-to-peer (P2P) note-taking application designed for
seamless data syncing across multiple devices without the reliance on a central
cloud server.
** Local Synchronization:
Sync notes directly between your devices,eliminating the need for
internet access and reducing dependency on external servers.
** Cross-Platform Compatibility:
Access your notes across all supported platforms—iOS, Android,
and desktops.
* Requirements
Mercury run over [[https://pears.com/][Pear]], to install run the next script
#+begin_src
npm i -g pear
#+end_src
* Install
#+begin_src
git clone https://github.com/nexhero/mercury.git
cd mercury

#+end_src
